### Search Endpoint
```
GET /search
```
Parameters:
- `country`: ISO 3166-1 country code
- `area`: (Optional) City/region name
- `tags`: Comma-separated list of OSM tags

### Countries Endpoint
```
GET /api/countries
```
Returns list of available countries with codes and languages.
